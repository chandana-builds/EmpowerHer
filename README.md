# 🌸 EmpowerHer – Smart Women Empowerment Platform

EmpowerHer is a modern digital platform designed to support women through **safety features, networking, mentorship, and opportunities** in a single unified system.

---

## 🚀 Features

- 🛡️ Safety alerts and emergency support  
- 🔐 Confidential ticket-based support system  
- 🤝 Networking and mentorship  
- 📚 Events and opportunities sharing  
- 🤖 AI-based assistance and recommendations  
- 🔑 Secure authentication with OTP  

---

## 🏗️ Tech Stack

- **Frontend:** React  
- **Backend:** Node.js  
- **Database:** MongoDB   
- **Deployment:** Vercel (Frontend), Render (Backend)  
- **APIs:** Email OTP, Maps Integration
